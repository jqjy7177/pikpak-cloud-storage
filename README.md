## 离线下载云盘是什么，和普通网盘有什么区别？

普通网盘的逻辑是：你在自己电脑上下好文件，再上传到云端存着。速度受你家宽带限制，上传大文件要等很久。

**离线下载云盘**的逻辑完全不同：你把一个磁力链接、BT 种子或者 HTTP 直链扔给它，它用自己的服务器去帮你下载，下完之后存到你的私人云盘里。你要看的时候，直接在线播或者再拉到本地。

好处是什么？服务器的带宽比你家宽带大多了。PikPak 官方说超过 80% 的文件能在几秒内完成云端下载——热门 BT 资源基本秒存，冷门资源速度慢一些但成功率也比本地下载高。有人试过一部本地挂了一周进度条纹丝不动的侯孝贤电影，丢给 PikPak 几秒就搞定了。

## 常见问题

**Q：PikPak 数据安全吗，会被查到下载记录吗？**

云下载发生在 PikPak 的远程服务器，通信使用加密协议，你的 ISP 无法监控你保存的内容。注册不需要实名，用 Google 账号登录就行。不过任何云服务都存在一定风险，涉及高度敏感内容时自行判断。

**Q：冷门资源能下吗？**

热门资源基本秒存（有人戏称 PikPak 是"迅雷新加坡分雷"，因为资源缓存量大）。冷门种子取决于资源热度，如果真的是死种，PikPak 也下不了。但很多本地下不动的老资源，用 PikPak 反而能成功，值得试试。

**Q：免费版空间只有 6GB，能做什么？**

临时下一部电影看完再删是完全够的（一部 1080P 电影大约 5-10GB）。周转用的话，下完同步到本地 NAS 或硬盘，再删掉腾空间，这个流程很多人在用。

**Q：月付还是年付划算？**

年付折合每月更便宜。如果只是偶尔用一次，可以先月付体验，觉得好用再换年付。区域会员的月付价格入门门槛比较低，适合先试试。

**Q：PikPak 支持 WebDAV 或挂载吗？**

官方不支持 WebDAV 上传，但可以通过 rclone 或 Alist 配合使用，将 PikPak 挂载为虚拟硬盘，方便跨平台访问。有一定技术门槛，适合折腾党。
